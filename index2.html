    Introdução


    HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas na Web. Ela estrutura o conteúdo que vemos em um site, como textos, imagens, links, listas, tabelas, vídeos, entre outros. Pense no HTML como o "esqueleto" da página — ele organiza os elementos que serão exibidos no navegador.


    Caracteristicas principais

    1. Não é linguagem de programação, pois não executa lógica.
    2. Não é linguagem de programação, pois não executa lógica.
    3. Usa tags (como h1, p, img) para identificar cada tipo de conteúdo.

    Tags

    As tags são os blocos fundamentais do HTML. Elas servem para marcar (ou "etiquetar") os diferentes tipos de conteúdo de uma página web — como títulos, parágrafos, imagens, listas, links e muito mais.

    Uma tag informa ao navegador qual o papel de cada parte do conteúdo.

    <p>Olá, Mundo</p>

    <p> abre uma tag e </p> fecha tag e "Olá, Mundo" é o conteúdo


    Tags comuns em HTML

    Tags Função
    <h1> a <h6> Títulos de diferentes níveis
    <p> Parágrafo de texto
    <a> Links
    <img> Inserção de imagem
    <ul>/<ol> Lista(não ordenado/ordenada)
    <li> Item da lista
    <div> Contêiner genérico de conteúdo
    <span> Contêiner em linha para texto

    Tags sem fechamento

    Algumas Tags não presisam de fechamento como: <img>, <br> e <hr> por exemplo.


    Resumo

    • Tags definem o tipo de conteúdo da página.
    • São envolvidas por < >
    • Usar tags corretamente é essencial para que o navegador entenda e exiba corretamente o conteúdo.
